# Character portraits

Drop portrait images here for the **Characters** tab on the campaign site.

| File | Character |
|------|-----------|
| `Aril000.png` | Aril (portrait slot) |
| `Inky000.png` | Inky (portrait slot) |
| `Kehan000.png` | Kehan (portrait slot — add when ready) |

Use numbered exports (`000`, `001`, …) as you render more poses. Full-body shots can get their own section later.

**Tips**
- Portrait orientation works best (~3:4 ratio, e.g. 520×693 px or larger).
- PNG with transparent background looks great against the dark frame.
- After adding files, commit and push — GitHub Pages updates in 1–2 minutes.

Until images are added, the site shows a subtle placeholder frame with the filename hint.
